# petNest
* Vue, typescript 공부 겸 만든 프로젝트

## Framework
* Vue.js Framework

## Library
* vue-router
* vue-property-decorator
* vuex
* style-loader
* ts-loader
* vue-loader
* css-loader
* vue-template-compiler
* 

```
npm install --save css-loader style-loader ts-loader vue-loader vue-template-compiler vue-router
```

## 개발계 서버 Start

```
npm run serve
```

또는

```
yarn serve
```
